# Roblox Convenience Store Social Learning MVP

장애아동의 사회성 학습을 돕기 위한 Roblox 기반 편의점 미션 시뮬레이션 MVP입니다.

## 프로젝트 목적

이 프로젝트는 편의점에서 물건을 고르고, 결제하고, 쿠폰을 사용하는 상황을 가상 환경에서 반복 연습할 수 있도록 설계되었습니다. 실제 편의점 이용 전, 사용자가 미션을 통해 사회적 상황과 절차를 익히는 것을 목표로 합니다.

## 주요 기능

- 제품 선택 미션
- 정해진 예산 안에서 물건 고르기
- 현금 결제 미션
- 카드 결제 미션
- 쿠폰/바코드 사용 미션
- NPC 점원 대화 시스템
- 미션 성공/실패 안내 UI

## MVP 테스트 목적

이번 MVP 테스트에서는 다음 항목을 확인합니다.

- 사용자가 미션 목표를 쉽게 이해하는지
- 조작 방식이 어렵지 않은지
- 결제 과정이 직관적인지
- 편의점 상황이 실제와 유사하게 느껴지는지
- 어느 부분에서 사용자가 헷갈리거나 막히는지

## 테스트 방법

1. Roblox 게임 링크에 접속합니다.
2. 화면 안내에 따라 미션을 진행합니다.
3. 플레이 후 피드백 폼을 작성합니다.
