## 프로젝트 개요
장애아동의 사회성 학습을 돕기 위한 Roblox 기반 편의점 훈련 시뮬레이션입니다.

## 프로젝트 목적

편의점에서 물건을 고르고, 다양한 결제수단을 사용하여 결제하고, 예산내에서 구매하여 거스름돈을 계산하고, 실제 편의점에서 발생할 수 있는 돌발상황에 대응할 수 있도록 설계하였습니다.

## 주요 기능

- 제품 선택 미션
- 정해진 예산 안에서 물건 고르기
- 현금 결제 미션
- 카드 결제 미션
- 쿠폰/바코드 사용 미션
- NPC 점원 대화 시스템
- 1+1/2+1 이벤트, 폰 화면 밝기 조절 등의 돌발상황 대응
